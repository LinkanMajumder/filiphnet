Futuristic UI (resources)

Hi, Filip here. The following are resources you might find useful if you're working on FUI (a.k.a. sci-fi UI). This list is meant to be the crème de la crème of the resources I have gathered over the years.

Some of these won't make sense unless you saw my WebExpo 2023 keynote talk Futuristic UI: How to make users feel like they’re in a sci-fi movie. I'm told the talk will be available online at some point.
